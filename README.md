# Express Geolocation and Weather API

This project sets up a basic web server using Express.js that exposes an API endpoint to return the client's IP address, location, and current temperature. It uses ipwhois.io for geolocation and OpenWeatherMap for weather data.

## Features

- **Geolocation**: Determines the city based on the client's IP address using ipwhois.io.
- **Weather Data**: Fetches the current temperature of the client's location using OpenWeatherMap.
- **API Endpoint**: Exposes a GET endpoint to retrieve the data.

## Requirements

- Node.js
- npm (Node Package Manager)
- AWS Account
- AWS CLI
- EC2 Instance (Amazon Linux 2 recommended)
